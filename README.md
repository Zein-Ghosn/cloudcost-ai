# CloudCost AI

CloudCost AI is an AI-powered AWS cost optimization platform built with a serverless architecture.

The system analyzes AWS Cost Explorer data and uses Amazon Bedrock to generate optimization insights for reducing cloud infrastructure costs.

## Live Demo

https://d9z4icievprc3.cloudfront.net/

## Architecture

Frontend
- React
- S3
- CloudFront

Backend
- API Gateway
- AWS Lambda

Data
- DynamoDB
- AWS Cost Explorer

AI
- Amazon Bedrock

Infrastructure
- Terraform

## Features

- AWS cost analytics dashboard
- AI-generated optimization insights
- Serverless architecture
- Infrastructure as Code with Terraform
